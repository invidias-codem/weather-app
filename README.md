# weather-app
A simple weather app that gives basic stats like humidity and wind speed using an API.

Features:

* Displays current weather conditions for a given location
* Search for locations
* Unit preferences (Celsius or Fahrenheit)
* Responsive design for various screen sizes


Technologies:

React: A JavaScript library for building user interfaces
OpenWeatherMap API: A free weather API
Axios: A promise-based HTTP client for making API requests
React Router: A routing library for single-page React applications
Material UI: A popular React UI library for creating visually appealing components

Getting Started

1. Clone this repository:

2. Install dependencies:

cd weather-app
npm install

3. Create an .env file in the root directory and add your OpenWeatherMap API key:

REACT_APP_OPENWEATHERMAP_API_KEY=your-api-key

4. Start the development server:

npm start

5. Open http://localhost:3000 in your browser.

Usage:

* Enter a location in the search bar and press Enter.
* The app will display the current weather conditions for that location.
* Click on the Settings icon to change the unit preferences.
